### How to run this really simple react frontend locally:

1 - You need to make sure you have Node.js installed. When you install it, npm comes with it. If you have Windows this tutorial shows you how to get them:
https://nodejs.org/en/download/

2 - Download the repo and save it somewhere.

3 - Run 'npm install' from the root of the repository.

4 - Run 'npm start' from the root of the repository. If it works it should display the simple text "Hello World" and on the next line "Hello from new!"

This example shows you how to export and import in a really simple way.
